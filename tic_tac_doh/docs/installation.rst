============
Installation
============


Stable release
--------------

To install tic_tac_doh, run this command in your terminal:

.. code-block:: console

    $ pip install tic_tac_doh

This is the preferred method to install tic_tac_doh, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for tic_tac_doh can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/psd314/tic_tac_doh

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/psd314/tic_tac_doh/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/psd314/tic_tac_doh
.. _tarball: https://github.com/psd314/tic_tac_doh/tarball/master
